== Description ==
The plugin is designed to import WP entries into Yandex Market.
Плагин предназначен для импорта записей WP в Яндекс Маркет. В ручном режиме обновление xml-файла
происходит на вкладке "Обновление" по нажатию кнопки.
Плагин распространяется "как есть". Данная версия плагина предназначена для получения обратной связи, в целях усовершенствования функционала плагина.


== Installation ==
Установить плагин, перейти по ссылке в меню на страницу настроек.
